# ONE STEP FORWARD FAKER

Script de génération de données fictives pour le remplissage de la base de données du projet "One Step Forward" (E-Santé & IoT)

---
## Commandes utiles

### Avant utilisation, installation des dépendances NPM : 

```
npm install
```

ou 

```
yarn
```

### Lancement du script 

```
npm start
```

Avec hot restarting :

```
npm run dev
```
